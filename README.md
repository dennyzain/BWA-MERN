## Installation

Install with yarn 

```bash
  yarn install
  cd dir
  yarn dev
```
    
## Features
- signUp
- signIn
- checkOut
- consume API store-dznx-API




## Tech Stack

**Client:** NextJS, Typescript, SSR, SSG, Bootstrap, JWTtoken

**Server:** Nodejs, Express, MongoDB
